# Nagad Cash Out Charge Calculator - Public Showcase

This repository is a **public showcase/sample version** of the Nagad Cash Out Charge Calculator project.

The full production source code is private. This repository only demonstrates the project idea, feature direction, privacy approach, and a small demo calculator so visitors can understand the work without getting the complete implementation.

## Live project

Production URL:

```text
https://nagad-calculator.web.app/
```

## What the full private project does

The private production project is a fast static website for Bangladeshi users who want to estimate Nagad cash out charges.

Production features include:

- Nagad cash out charge calculation
- App, Islamic App, and USSD method support
- Bangla and English language support
- Bangla and English digit input support
- Instant client-side result updates
- Mobile-first responsive design
- SEO-friendly content structure
- FAQ and educational sections
- About, Privacy Policy, Terms, Contact, and Disclaimer pages
- Sitemap and robots.txt setup
- Firebase Hosting deployment
- AdSense-friendly policy structure
