# Deployment Notes

The private production project is deployed separately and is not represented by this public showcase repository.

## Production hosting

The production version is intended for Firebase Hosting.

## Public showcase hosting

This showcase can be opened locally by visiting:

```text
demo/index.html
```

It is not meant to replace the full production website.

## Do not publish private files

Before pushing any repository publicly, avoid uploading:

- Full production source code
- `.firebase/` cache files
- `.env` files
- AdSense private setup details
- Search Console verification files unless intentionally public
- Any API keys or credentials
