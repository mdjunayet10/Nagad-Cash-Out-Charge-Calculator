# Security and Privacy Notes

The production project is designed as a client-side informational calculator.

## What the app does not collect

- Nagad PIN
- OTP
- Password
- Account number
- Personal financial information
- Transaction credentials

## Privacy approach

The calculator works in the browser and does not require login. Users can calculate estimated fees without sharing sensitive information.

## Brand safety

The project avoids using official brand assets unless legally permitted. The public showcase uses an original simple icon and does not claim official affiliation.

## Disclaimer

This project is independent and informational only. Users should verify final charges from official Nagad channels.
